#  Free Game Website
A website that's dedicated to finding free games 

## Installation
```npm install```

## Dev Mode
```yarn dev```

## Build
```yarn build```

## Add Keys
Add a .env file and add
```REACT_APP_API_KEY```
```REACT_APP_API_HOST```
